# Predictor

Streamlit UI for the bus/legua door-hanging safety classifier.
Upload an image; the app preprocesses it (matching `model/preprocess.py`),
extracts HOG features, loads a trained sklearn pipeline from
`../model/*.joblib`, predicts the class, and shows a **LIME** region-importance
heatmap explaining which parts of the image drove the decision.

The poetry env is scoped to this folder only.

## Labels

| Label | Class | Meaning |
|---|---|---|
| `0` | negative (safe) | Passengers are NOT hanging on the door |
| `1` | positive (UNSAFE) | Passengers ARE hanging on the door |

This follows the standard ML convention (positive = event detected) and the
class ordering used by the saved joblib models.

## One-time setup

```bash
cd predictor

# 1) Make sure poetry is available (installed via `pip install --user poetry`).
#    If `poetry` is not on PATH, use `python3 -m poetry` instead of `poetry`.

# 2) Keep the virtualenv inside this folder
python3 -m poetry config virtualenvs.in-project true --local

# 3) Install dependencies (creates predictor/.venv and predictor/poetry.lock)
python3 -m poetry install --no-root
```

## Run the app

```bash
python3 -m poetry run streamlit run app_streamlit.py
```

Opens at <http://localhost:8501>. Sample images live under `predictor/images/`.

In the sidebar you can toggle the LIME heatmap and tune the number of
perturbation samples (more = smoother map, slower).
